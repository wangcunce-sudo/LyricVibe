# LyricVibe 产品方案简介

> AI 个性化歌词字幕 MV 生成器 — 让每个人都成为 MV 导演

---

## 一、产品定位

**一句话描述：** LyricVibe 是一个面向大众用户的 AI 歌词字幕 MV 生成工具，用户只需上传视频和音频，AI 自动理解歌词情感、生成匹配的字幕样式，一键导出高质量 MP4。

**目标用户：** 普通大众 — 社交媒体用户、短视频创作者、K 歌爱好者、Vlog 博主，无需任何视频编辑经验。

**产品形态：** Web 应用（浏览器即用）+ 可选 Docker 私有部署。

---

## 二、问题洞察

### 2.1 市场现状

| 工具 | 定位 | 痛点 |
|---|---|---|
| **剪映 / CapCut** | 专业视频编辑 | 模板固定、无 AI 歌词理解、需要手动调整时间轴 |
| **Neural Frames** | AI 音乐可视化 | 面向音乐人、需付费、不支持用户上传视频、学习门槛高 |
| **Kapwing / Veed** | 在线视频编辑 | 字幕功能简单、无歌词情绪分析、按使用付费 |
| **手动做歌词 MV** | Premiere/AE 等 | 耗时数小时、需要设计 + 剪辑技能 |

### 2.2 核心痛点

1. **歌词字幕太"死板"** — 现有工具的字幕只有位置/字体/颜色变化，无法根据歌词内容智能匹配风格。一段热舞和一段抒情歌用同样的字幕样式，情感表达错位。

2. **时间轴对齐极度繁琐** — 逐词对齐需要手动拖拽时间轴，一首 3 分钟的歌曲可能需要 1-2 小时对齐。

3. **没有设计能力** — 普通用户不知道什么字体、颜色、动画搭配好看，也写不出好的字幕效果描述。

4. **工具分散** — 需要先用语音识别工具转歌词，再用字幕工具做时间轴，再用视频编辑器合成，流程断裂。

### 2.3 机会窗口

- TikTok/Instagram Reels/YouTube Shorts 等短视频平台爆发式增长，歌词字幕 MV 是最受欢迎的内容类型之一
- AI 大模型能力成熟（GPT-4/Claude/DeepSeek），可以深度理解歌词语义和情感
- WhisperX 等开源语音识别模型精度已达商业可用水平
- Remotion 等 React 视频渲染框架成熟，可以复用 Web 前端组件直接渲染视频

## 三、解决方案

### 3.1 核心流程

```
上传视频 + 音频  →  WhisperX AI 歌词识别  →  AI 情感分析 + 风格生成  →  实时预览  →  一键导出 MP4
     ↓                    ↓                          ↓                       ↓                ↓
  支持拖拽           Demucs 音源分离            情绪/主题/节奏分析        Canvas 实时渲染    Remotion SSR
  任意格式          large-v3 高精度转录         自动匹配字体/颜色/动画    所见即所得          1080p 输出
```

### 3.2 技术亮点

| 能力 | 技术实现 | 用户价值 |
|---|---|---|
| **AI 歌词情感分析** | OpenAI / DeepSeek / Anthropic 多 provider 支持 | 理解歌词情绪（能量/喜悦/忧伤等 8 种），自动生成匹配风格 |
| **自然语言生成字幕** | LLM 将自然语言描述转化为渲染参数 | "深色哥特风衬线体，血红强调色" → 即时生成可用的字幕样式 |
| **高精度时间轴** | WhisperX（Demucs 音源分离 + faster-whisper large-v3） | 逐字级别时间戳，支持逐词弹跳动效 |
| **前后端同构渲染** | Remotion 4 + React 19 | 前端预览 100% 等于最终导出效果，无"所见非所得"问题 |
| **实时音频处理** | Tone.js 时间拉伸 + 变调 | 0.5x–2x 变速 ±12 半音变调，字幕自动同步 |
| **多种字幕动画** | 7 种动画 + 8 种视频滤镜 | kinetic-pop / fade-in / karaoke / typewriter 等 |
| **多 provider 架构** | AI 服务抽象层 | 支持 OpenAI / DeepSeek / Anthropic 自由切换，不绑定供应商 |

### 3.3 产品差异化

| | 剪映/CapCut | Neural Frames | **LyricVibe** |
|---|---|---|---|
| **AI 歌词理解** | ❌ | ❌ | ✅ 8 种情绪分析 |
| **自然语言风格生成** | ❌ | ❌ | ✅ 说话就能调样式 |
| **上传自有视频** | ✅ | ❌ | ✅ |
| **WhisperX 逐词时间轴** | ❌ | ❌ | ✅ |
| **学习门槛** | 中 | 高 | **零** |
| **价格** | 免费（模板付费） | $19/月起 | **开源免费** |

## 四、产品设计

### 4.1 功能架构

```
LyricVibe
├── 🏠 首页
│   ├── 产品介绍 + 创作流程展示
│   ├── Demo 视频预览（成品展示）
│   └── 一键进入创作工作台
├── 🎬 创作工作台
│   ├── 左侧：视频预览区
│   │   ├── Remotion Player 实时预览
│   │   ├── Canvas 字幕叠加渲染
│   │   ├── 播放/暂停/进度控制
│   │   └── 变速/变调/静音控制
│   ├── 右侧：控制面板（4 个 Tab）
│   │   ├── 🎨 风格 Tab
│   │   │   ├── AI 情感分析结果（情绪雷达 + 主题标签）
│   │   │   ├── 风格提示词（可编辑 + 重新生成）
│   │   │   ├── 🤖 AI 字幕模板生成（自然语言输入）
│   │   │   ├── 🎨 字幕模板预设（5 种预制风格）
│   │   │   └── 快捷模板 + 风格变体
│   │   ├── 🎞️ 滤镜 Tab（8 种视频滤镜）
│   │   ├── 🎵 音频 Tab（变速 0.5x–2x + 变调 ±12）
│   │   └── ⚙️ 高级 Tab（分辨率/帧率/画质等）
│   └── 顶部操作栏
│       ├── 返回首页 / 体验演示 / 中英文切换
│       └── 一键导出（Remotion SSR 服务端渲染）
└── 🔧 API 服务
    ├── /api/upload      — 文件上传
    ├── /api/transcribe  — WhisperX 歌词识别
    ├── /api/analyze     — AI 情感分析 + 风格生成
    ├── /api/template    — 自然语言 → 字幕模板
    └── /api/render      — Remotion 视频渲染
```

### 4.2 用户体验设计原则

1. **零学习成本** — 无需教程，拖入视频即可开始
2. **渐进式复杂度** — 默认一键出片，高级用户可深入调整每个参数
3. **即时反馈** — Canvas 实时预览，修改风格/滤镜/速度立即可见
4. **Demo 驱动** — 内置完整 Demo，无需上传素材即可完整体验

### 4.3 Demo 模式

内置 `opalite-tiktok热舞` 演示内容：
- 32 秒 TikTok 风格热舞视频 + 背景音乐
- AI 预分析结果（energy 95%、triumph 88%）
- 蓝白弹跳字幕模板（Impact 粗体 + 发光效果）
- 用户可直接预览、修改风格、一键导出体验完整流程

## 五、商业模式

### 5.1 开源免费（当前阶段）

- **代码开源**（MIT License）— GitHub 可自由使用、修改、分发
- **自部署** — Docker 一键部署到自己的服务器
- **API Key 自理** — 用户自行申请 AI 服务商 API Key

### 5.2 潜在商业化方向

| 模式 | 说明 |
|---|---|
| **SaaS 托管版** | 免部署，网页直接使用，按月/按次付费 |
| **高级模板市场** | 专业设计师上传字幕模板，平台抽成 |
| **企业私有部署** | 为企业提供定制化部署 + SLA 保障 |
| **API 服务** | 将字幕生成能力作为 API 输出给第三方 |

## 六、技术栈

| 层级 | 技术 | 选型理由 |
|---|---|---|
| **前端框架** | Next.js 16 (App Router) | 全栈能力，SSR + API Routes 一体化 |
| **UI** | React 19 + Tailwind CSS 4 | 现代化组件 + 原子化样式 |
| **视频渲染** | Remotion 4 | React 组件直接渲染为视频，前后端同构 |
| **AI 分析** | OpenAI / DeepSeek / Anthropic | 多 provider 可切换，推荐 DeepSeek（性价比最高） |
| **语音识别** | WhisperX（Python） | 业界最佳开源方案，Demucs 音源分离 + large-v3 转录 |
| **音频处理** | Tone.js | 浏览器端实时时间拉伸 + 变调 |
| **Canvas 渲染** | HTML5 Canvas API | 字幕叠加到视频上的高性能预览 |
| **类型安全** | TypeScript 5 + Zod | 编译时类型检查 + 运行时输入验证 |
| **国际化** | 自研 i18n | 中英文完整支持 |
| **部署** | Docker + docker-compose | 一键部署，环境一致 |
